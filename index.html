Features

Everything You Need

Powerful features designed for professionals who need efficient image gallery management.

🔄

Sequential Multi-Page

Processes pages one at a time - scans, downloads all images, then automatically moves to the next page.

⏸

Pause & Resume

Pause long operations anytime and resume from checkpoints saved every 5 pages. Never lose progress.

⚡

Adaptive Timing

Dynamic wait times based on site speed. 40-60% faster on responsive sites, more reliable on slower ones.

💾

Memory Monitor

Automatic memory monitoring and pausing to prevent crashes when handling large galleries (5000+ images).

🔔

Toast Notifications

User-friendly error messages and visual feedback for all operations. No confusing technical errors.

📄

7 Pagination Methods

Next button, load more, infinite scroll, arrow navigation, URL patterns, API-based, and auto-detect.

📊

CSV Export

Export image data to CSV format for spreadsheets with customizable field selection.

⬇

Batch Downloads

Download all images with custom filename patterns and concurrent download management.

🔒

Privacy First

No external API calls, no tracking, no analytics. All processing happens locally in your browser.

✅

MV3 Compliant

Built for Chrome 2025 with Manifest V3 for future-proof compatibility and security.

Installation

Quick Start Guide

Get up and running in less than a minute.

1

Download the Extension

Download all files from this repository to your local machine.

2

Open Chrome Extensions

Navigate to chrome://extensions/ in your Chrome browser.

3

Enable Developer Mode

Toggle the "Developer mode" switch in the top-right corner of the page.

4

Load Unpacked Extension

Click "Load unpacked" and select the StepGallery folder containing manifest.json.

5

Start Using

The extension icon should appear in your toolbar. Click it to open the side panel!

Usage

How It Works

Simple workflow for maximum productivity.

Basic Workflow

  • 1 Navigate to any webpage with an image gallery
  • 2 Click the StepGallery icon in your Chrome toolbar
  • 3 Review the gallery detection results in the side panel
  • 4 Select a pagination method or use "Auto-Detect"
  • 5 Start pagination to collect images across pages
  • 6 Export data or download all images

Pro Tips

  • 💡 Use Auto-Detect for best results on most sites
  • 💡 Increase pagination delay for slow-loading galleries
  • 💡 Use batch confirmation for large downloads
  • 💡 Pause anytime and resume from checkpoint
Customization

Filename Tokens

Customize your downloaded filenames with powerful tokens.

*num* Image number (1, 2, 3...)
*num-3* Padded number (001, 002...)
*name* Original filename
*ext* File extension
*date* Current date
*time* Current time
*page* Page number
*caption* Image caption/alt
*domain* Website domain
